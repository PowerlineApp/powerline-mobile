js-collection
=============

Javascript collection and model

### JsModel
 - set / get
 - parsers (may use for tree like structures, dates, etc.)
 - default parsers: 'date'
 - extend for custom methods

### JsCollection
 - model class
 - extend for custom methods
 - comparator
 - lodash methods
